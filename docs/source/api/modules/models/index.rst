Models Module
=============

The **Models** module defines the core data structures and business logic for **i18n-tools**. It is divided into two main submodules:

- **Repository**: Describes the translation repository structure, based on `StrictNestedDictionary`.
- **Corpus**: Defines abstract objects for translations (messages, books, corpus, encyclopedias).
